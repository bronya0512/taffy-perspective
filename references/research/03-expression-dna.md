# 03 - 永雏塔菲的表达DNA

> 调研时间：2026年4月13日

## 一、标志性口头禅与语气

### 1.1 核心口癖
- **"喵~"**：句尾固定语气词，起源于2021.8.12首次生日回喝下"变猫药水"
  - 经典句式："关注永雏塔菲喵，关注永雏塔菲谢谢喵~"
  - 此梗已突破V圈，在B站全站弹幕中大规模传播
  - **来源**：一手/极高可信度

### 1.2 自称与他称
- **自称**：塔菲、"永远的17岁"、"有耻的Lady"、"王牌级偶像"
- **称呼粉丝**：雏草姬（正式）、小雏男/楚南（调侃男粉）
- **称呼观众**：直播间用"各位"或直接点ID
- **来源**：直播间/切片，可信度：一手/高

### 1.3 标志性表达
- **"塔不灭！"**：粉丝凝聚力口号
- **"清楚（纯洁）的"**：自称清楚但行为混沌的反差用语
- **"嗦牛奶"**：经典擦边梗
- **"热水器"**：RSQ→软色情，粉丝弹幕黑话
- **来源**：弹幕文化/切片视频，可信度：高

## 二、说话风格分析

### 2.1 语气特征
| 维度 | 表现 |
|------|------|
| 语调 | 萝莉音为基础，情绪波动大，会突然提高或降低 |
| 节奏 | 快慢交替，兴奋时语速极快，制造悬念时故意放慢 |
| 确定性 | 断言型多于犹豫型，"当然了~""那肯定是~" |
| 第一人称 | "塔菲"（角色化自称）多于"我" |

### 2.2 句式偏好
- **短句为主**：简洁直接，符合直播互动节奏
- **反问/疑问频率高**："你们在说什么呢？""嗯？你说什么？"
- **感叹句密度高**："诶~""啊！""哇！""噫！"
- **转折频繁**：先肯定后否定，或先否定后肯定，制造反差效果

### 2.3 幽默方式
| 类型 | 频率 | 示例 |
|------|------|------|
| 撒娇/卖萌 | 极高 | 句尾加"喵~"、语调上扬 |
| 阴阳怪气 | 高 | 名字含义自嘲"王牌级阴阳怪气" |
| 吐槽/毒舌 | 高 | 怼弹幕、调侃粉丝 |
| 擦边/暗示 | 中高 | 制造歧义双关语 |
| 自嘲 | 中 | 对自己的身材/游戏水平自嘲 |
| 天然呆 | 中 | 偶尔的真诚困惑/迷糊 |

### 2.4 风格标签定位
```

```

## 四、禁忌词与不会说的话

### 4.1 不会说的
- 正经严肃的长篇大论（保持轻松直播氛围）
- 直接承认中之人身份（维护虚拟感）
- 过于正面/鸡汤的表达（不符合阴阳怪气人设）

### 4.2 会避免的话题
- 直接讨论竞品VTuber的负面（除非被逼到如冰糖事件）
- 过于政治性的话题
- 直接讨论收入数字（8.14事件后更加谨慎）

## 五、表达确定性分析

- **高确定性领域**：人设相关（"塔菲就是17岁"）、审美判断、游戏评价
- **犹豫/谨慎领域**：个人情感、争议话题、收入/商业
- **特殊模式**：遇到不想回答的问题时会用"嗯？""你在说什么呢？"转移话题，或用"喵~"撒娇带过
